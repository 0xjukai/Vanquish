# Vanquish
Root2Boot automation platform designed to systematically enumerate and exploit using the law of diminishing returns (based on Pillage).
Includes :
  * Nmap Scanning
  * GoBuster
  * Nikto
  * SSH
  * mySQL
  * MSSql
  * RDP
  * SMB
  * SMTP
  * SNMP
  * SSH
  * FTP
  * DNS
  * Web

Usage:

python Vanquish.py <hostfile.txt>

Example:

python Vanquish.py hosts.txt
